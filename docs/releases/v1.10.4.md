## 修复

- 修复 Windows 打开超长 JSON 后点击“展开全部”会递归创建数千个树节点、导致窗口进入“未响应”的问题；大文档现在只展开顶层，具体数组仍可按需展开和分页查看。
- 修复超长 JSON 虚拟编辑器在 Windows 上显示成多个带圆角边框的分段输入框；保留原来的连续文本外观、完整滚动、复制和直接修改能力。
- 修复“关于”窗口发布日期长期固定为 `2026-07-14` 的问题；发布日期现在从安装包版本中的构建日期自动解析。

## 优化

- 超长编辑器不再预构建屏幕外文本输入组件，并将区块索引更新从逐块扫描改为常数时间查找，减少 Windows 文本输入与辅助功能树负担。
- 大文档搜索改到后台 isolate 执行，并限制一次保留的匹配路径数量；树视图只展开当前匹配路径，不再为每个可见节点重复扫描整个子树。
- Windows 发布流水线新增超长 JSON 专项回归，覆盖 10 MB 文本拆分、连续编辑、受限树展开、后台搜索和版本日期解析。
