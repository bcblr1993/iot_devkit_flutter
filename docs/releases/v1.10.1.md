## 新增

- 自定义 Key 新增单项“生效/停用”开关，停用后保留完整配置，但不参与数据生成和 MQTT 发送，后续可直接重新启用。
- 基础模式新增自定义 Key 总开关；高级模式为每个分组提供独立总开关。关闭总开关不会改写任何单项状态，重新开启后会恢复原有选择。
- 自定义 Key 区域新增生效数、保留数及“待生效”状态提示，便于区分当前发送项和暂存配置。

## 修复

- 修复已停用自定义 Key 仍占用键总数、参与重复名称和取值范围校验的问题。
- 修复关闭总开关后，预览、配置校验与实际 MQTT 发送链路行为不一致的问题。
- 修复高级模式分组表单因字段数量变化产生的空白格和数据格式控件错位，将身份字段整理为两行三列，并将数据格式恢复为完整行布局。

## 兼容性

- 旧配置没有开关字段时，单项开关和总开关均默认开启，升级后保持原有发送行为。
- 导入、导出和自动保存会保留单项开关以及基础模式/高级分组的总开关状态。
