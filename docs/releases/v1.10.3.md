## 修复

- 修正 v1.10.2 对超大 JSON 只显示限量预览的问题，恢复为完整、可滚动、可编辑的文本编辑器。
- 修复普通多行输入框一次布局整份 10 MB 文本导致界面卡死的问题；超大文档改为分块保存并按可见区域虚拟化渲染。
- 修复大段粘贴可能先进入单个输入框并触发昂贵布局的问题，超过阈值的输入会在渲染前直接拆分为可编辑区块。

## 优化

- 所有区块按原始顺序和换行符无损拼接，格式化、压缩、树形解析及工具栏复制始终处理完整文档。
- 编辑器只创建屏幕附近的少量输入框，滚动到任意位置后均可直接修改；区块边界支持退格和删除合并。
- 压缩成单行后的超长 JSON 同样按字符分块显示，不会重新退化为单个超大文本布局。

## 验证

- 使用实际 10,745,012 字符、492,001 行 JSON 的 macOS Release 应用验收：粘贴约 0.5 秒进入完整编辑器，树形视图正确显示 30 个键及 122,985 条记录。
- 已验证连续滚动可加载文档后部、可见区块可直接修改；压缩后的 4,595,370 字符仍可滚动和编辑。
- 自动化测试覆盖 10 MB 文本无损拆分/还原、只构建可见区块、滚动到最后一块修改末尾内容，以及完整文档复制。
- 已通过 Flutter 静态分析、Lab 自定义规范检查、Widget/服务测试和 macOS Release 构建验收。
