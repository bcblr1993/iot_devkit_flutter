## 新增

- **MQTT 订阅**：模拟设备连接成功后可订阅任意 topic（支持 `+` / `#` 通配符），收到的消息实时进入日志控制台（前缀 `[← topic]`，超长 payload 自动截断）。
- **ThingsBoard 预设**：一键添加 `v1/devices/me/rpc/request/+`（RPC）与 `v1/devices/me/attributes`（共享属性）两条订阅。
- **RPC 自动回包**：收到 ThingsBoard 服务端 RPC 请求时，可自动向 `v1/devices/me/rpc/response/{id}` 回一个空响应 `{}`（可单独开关）。
- **启用订阅开关**：与「启用 SSL/TLS」同行的主控开关，关闭后运行时不下发任何订阅；订阅配置随模拟配置导入导出持久化（旧配置无该字段时自动兼容）。
- 订阅相关文案与主题选择器描述标签均已国际化（中 / 英）。

## 优化

- 订阅配置内嵌进「MQTT 代理配置」面板，与连接设置同区，语义更清晰。
- 模拟统计指标（设备数 / 在线 / 已发送 / 成功 / 失败 / CPU / 内存）移入日志控制台底栏，常驻可见且横向自适应。
- 证书生成、工时管理两个页面改为满宽自适应，宽窗口下不再两边留白，与其它页面一致。
- 接入 UI 一致性三层防线：静态规则（自定义 lint：颜色 / 间距 / 圆角）+ golden 视觉回归 + widget 冒烟。

## 修复

- 修复同时开启「SSL/TLS」与「订阅」时，MQTT 面板开关行裁剪导致证书字段标签被遮挡、间距错乱的问题。
- 修复证书生成 / 工时管理页面在宽窗口下两边出现空白、未自适应窗口的问题。
